# Principais Comandos

- git init - Inicializa um repositório
- git config --global user.email" " - configura o seu email no git
- git config --global user.name - configura o nome de usuário que vai estar nos projetos
- git add * - adiciona os arquivos novos e mudanças para serem commitados
- git commit -m "anotação referente as mudanças" - commita o repositório
- git status - retorna a situação do repositório
- git config --list - mostra as configurações salvas no git
- git push origin master - envia o repositório atualizado para o github
- git pull origin master - puxa o repositório atualizado mais recente do github para o repositório local
- git clone url do github - para clonar repositório
